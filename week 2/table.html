HTML Tables — Tags and Attributes

This page shows how to build tables using the <table>, <tr>, <th> and <td> tags and how to use the attributes border, colspan and rowspan.

1. Simple table (using border attribute)

The border attribute (legacy) draws a border around cells. Modern pages typically use CSS, but this demonstrates the attribute.

The colspan attribute makes a cell span multiple columns.

The rowspan attribute makes a cell span multiple rows.

Notes

  • <table> defines the table.
  • <tr> defines a table row.
  • <th> defines a header cell (usually bold and centered).
  • <td> defines a regular data cell.
  • colspan and rowspan merge cells horizontally and vertically respectively.

Try changing the numeric values of colspan and rowspan to see how the table layout changes.
